# Mishka Memory

## Описание
Сервис долгосрочной памяти. Управляет пользователями, историей чатов (Redis/Postgres) и фактами (Qdrant).

## Архитектура
*   **Inputs**:
    *   HTTP API
*   **Outputs**:
    *   PostgreSQL (Пользователи)
    *   Redis (Кеш истории)
    *   Qdrant (Векторная база фактов)

## Переменные окружения (Static)
| Переменная | Описание |
| :--- | :--- |
| `POSTGRES_*` | Параметры подключения к БД. |
| `REDIS_*` | Параметры Redis. |
| `QDRANT_*` | Параметры Vector DB. |

## Динамические настройки (Dynamic)
Управляются через **Admin Panel**.

| Ключ | Тип | Описание |
| :--- | :--- | :--- |
| `context_limit` | int | Сколько последних сообщений хранить в "быстрой" памяти (Redis). |
