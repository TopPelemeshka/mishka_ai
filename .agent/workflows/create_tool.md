# Создание инструмента

1. **Спроси пользователя:**
   - Название инструмента (на английском, snake_case).
   - Описание (что он делает).
   - Какие внешние API используются (если есть).

2. **Создай файл:**
   - Путь: `src/core/tools/{tool_name}.py`
   - Содержимое:
     - Класс, наследуемый от `BaseTool`.
     - Pydantic модели для входных параметров (`InputSchema`) и настроек (`SettingsSchema`).
     - Метод `run` с заглушкой логики и обработкой ошибок.
     - Docstring с описанием.

3. **Обнови реестр:**
   - Если есть файл реестра (например `src/core/tools/__init__.py`), добавь туда импорт.

4. **Предложи:**
   - Сразу написать unit-тест для этого инструмента.
