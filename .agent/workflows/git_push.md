---
description: Автоматизирует коммит и пуш изменений с генерацией сообщений на русском языке.
---

1. **Проверка статуса**:
   - Выполни `git status --porcelain`.
   - Если вывод пуст, ответь "Изменений не найдено." и остановись.

2. **Анализ изменений**:
   - Выполни `git diff`, чтобы увидеть измененные файлы.
   - Если есть неотслеживаемые файлы (untracked), выполни `git status`, чтобы узнать их имена.
   - Прочитай содержимое измененных/новых файлов, если это необходимо для понимания контекста.

3. **Генерация сообщения**:
   - Сгенерируй сообщение коммита в формате **Conventional Commits**, но на **РУССКОМ ЯЗЫКЕ**.
   - Формат: `тип(область): описание` (например, `feat(auth): добавлена поддержка базы данных` или `fix(api): исправлен таймаут соединения`).
   - Сообщение должно быть кратким и четким.

4. **Запрос подтверждения**:
   - Используй `notify_user`, чтобы показать сообщение:
     > Я предлагаю следующее сообщение коммита:
     >
     > `[сгенерированное_сообщение]`
     >
     > Выполнить `git add .`, commit и push? (y/n)

5. **Выполнение**:
   - **Жди** ответа пользователя "Yes" или "y".
   - Если одобрено, выполни:
     ```bash
     git add .
     git commit -m "[сгенерированное_сообщение]"
     git push
     ```
   - Используй `git push origin main` или текущую ветку, если она известна.
